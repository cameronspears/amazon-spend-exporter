# Code of Conduct

## Our pledge
We are committed to a welcoming, respectful, harassment-free community.

## Expected behavior
- Be respectful and constructive.
- Assume good intent, ask clarifying questions.
- Focus criticism on code and ideas, not people.

## Unacceptable behavior
- Harassment or discriminatory language.
- Personal attacks or intimidation.
- Publishing others' private information.

## Enforcement
Project maintainers may remove comments, close issues, or ban contributors who violate this code.

## Reporting
Open a private security report as described in [SECURITY.md](./SECURITY.md) for sensitive incidents.
